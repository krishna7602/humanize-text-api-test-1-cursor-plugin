## 🔌 How to Install

### Option A: IDE Integration (Cursor Local)
1. Open **Cursor Settings** (`Ctrl+Shift+J` or `Cmd+Shift+J`).
2. Navigate to **Features** > **Tools & MCP**.
3. Click **+ New MCP Server**.
4. Configure as follows:
   - **Name**: `Humanize Text API test 1`
   - **Type**: `sse`
   - **URL**: `https://humanize-text-fkrzt36pn-r-krishna-mondals-projects.vercel.app/sse`

### Option B: Cloud Agents Integration
1. Go to [cursor.com/agents](https://cursor.com/agents).
2. Click **Add Tools** or the **+** icon in the MCP section.
3. Input the SSE URL: `https://humanize-text-fkrzt36pn-r-krishna-mondals-projects.vercel.app/sse`

## 📁 Project Structure
- `.cursor-plugin/`: Plugin metadata for the Cursor Marketplace.
- `skills/`: Individual `SKILL.md` instruction files giving precise routing signals to Cursor's model for each tool.
- `.cursorrules`: Instruction set for Cursor AI.
- `mcp.json`: Connection manifest for SSE server.
